# Session 1: Foundation & UI Architecture

## 목표
프로젝트의 뼈대를 세우고 직무/레벨 선택 흐름을 완성합니다.

## Task 1: Project Initialization & Structure

Vite + React + Tailwind CSS 프로젝트를 초기화하고 폴더 구조를 세팅해주세요.

### 요구사항
- `npm create vite@latest . -- --template react` 로 프로젝트 생성
- Tailwind CSS v3 설치 및 설정 (`tailwind.config.js`, `postcss.config.js`)
- 폴더 구조 생성:
  ```
  src/
    components/
      common/       # Button, Card, Layout 등 공통 컴포넌트
      roadmap/      # 마인드맵 관련 컴포넌트
      selection/    # 직무/레벨 선택 컴포넌트
    hooks/          # 커스텀 훅
    services/       # LLM, API 서비스
    utils/          # 유틸리티 함수
    context/        # React Context
    data/           # 로컬 데이터
  public/
    data/           # JSON 데이터 파일
  ```
- 공통 UI 컴포넌트 작성:
  - `Button.jsx` - 기본/강조/아웃라인 variant 지원
  - `Card.jsx` - 클릭/선택 상태 지원
  - `Layout.jsx` - 페이지 기본 레이아웃 래퍼

### 커밋 메시지
```
feat: 프로젝트 초기화 및 공통 컴포넌트 구축

- Vite + React + Tailwind CSS 환경 설정
- components/, hooks/, services/, utils/, context/ 폴더 구조 생성
- Button, Card, Layout 공통 UI 컴포넌트 작성
```

## Task 2: Role & Level Selection Flow

직무(Role) 및 숙련도(Level) 선택 카드 UI와 전역 상태 관리를 구현해주세요.

### 요구사항
- `SelectionContext.jsx` - 선택 상태 전역 관리 (React Context + useReducer)
  - 상태: `{ role: null, level: null }`
  - 액션: `SET_ROLE`, `SET_LEVEL`, `RESET`
- `RoleCard.jsx` - 직무 선택 카드
  - 직무 목록: Backend, Frontend, AI/ML, DevOps, Mobile, Data Engineering
  - 각 직무별 아이콘(이모지 또는 SVG), 설명 텍스트 포함
  - 선택 시 강조 테두리 + 체크 표시
- `LevelCard.jsx` - 숙련도 선택 카드
  - 레벨: Junior (0-2년), Mid (2-5년), Senior (5년+)
  - 레벨별 특징 설명 포함
- `SelectionPage.jsx` - Role → Level 2단계 선택 흐름
  - Role 선택 후 Level 선택으로 자연스럽게 전환
  - 선택 완료 시 "로드맵 생성하기" 버튼 활성화

### 커밋 메시지
```
feat: 직무/레벨 선택 UI 및 전역 상태 관리 구현

- SelectionContext로 role, level 전역 상태 관리
- RoleCard, LevelCard 선택 컴포넌트 작성
- SelectionPage에서 2단계 선택 흐름 구현
```

## Task 3: Visual Engine Layout (SVG)

계층적 SVG 마인드맵 레이아웃 엔진을 구현해주세요.

### 요구사항
- `MindmapLayout.jsx` - 왼쪽(Core) → 오른쪽(Leaf) 방향의 위계적 SVG 레이아웃
  - 노드 위치 계산 알고리즘: 루트 노드 좌측 고정, 자식 노드 우측으로 펼침
  - 노드 간 연결선 (Bezier Curve 권장)
  - 노드 클릭 이벤트 핸들러 (onNodeClick prop)
  - 노드 데이터 구조:
    ```js
    {
      id: string,
      label: string,
      level: number,       // 트리 깊이 (0=root, 1=branch, 2=leaf)
      children: Node[],
      status: 'locked' | 'available' | 'completed'
    }
    ```
- `MindmapNode.jsx` - 개별 노드 컴포넌트
  - 레벨별 스타일 차별화 (루트: 큰 원, 브랜치: 중간 사각형, 리프: 작은 타원)
  - status에 따른 색상 변화 (locked: 회색, available: 파란색, completed: 초록색)
- `ListView.jsx` - 동일 데이터를 아코디언 목록으로 표시
- `ViewToggle.jsx` - 마인드맵/리스트 뷰 전환 토글 버튼
- 더미 데이터로 레이아웃 동작 확인

### 커밋 메시지
```
feat: SVG 마인드맵 레이아웃 엔진 및 리스트뷰 구현

- 계층적 SVG 마인드맵 노드 배치 알고리즘 구현
- MindmapNode 컴포넌트에 status별 스타일 적용
- ListView 아코디언 컴포넌트 및 뷰 전환 토글 추가
```
