## 진행 현황 (중단 시점)

### ✅ 완료된 작업

#### 백엔드
- `backend/app/models/schemas.py` — 신규 스키마 추가 완료
  - `NodeDetailRequest`, `NodeDetailResponse`
  - `QuizRequest`, `QuizResponse`, `QuizQuestion`, `QuizOption`
  - `ChatRequest`, `ChatMessage`, `NodeContext`
- `backend/app/utils/prompts.py` — 신규 프롬프트 함수 추가 완료
  - `build_node_detail_prompt(node_label, role, level)` → `(system, user)` 튜플 반환
  - `build_quiz_prompt(node_label, role, level)` → `(system, user)` 튜플 반환
  - `build_chat_system_prompt(node_label, role, level)` → system 프롬프트 문자열 반환
- `backend/app/routers/node.py` — 신규 라우터 파일 생성 완료
  - `POST /api/node/detail` — 미션/개념 생성 + 인메모리 캐시 (`_detail_cache`)
  - `POST /api/node/quiz` — OX/객관식 퀴즈 3문제 생성 + 인메모리 캐시 (`_quiz_cache`)
  - 캐시 키: `{role}_{level}_{node_label}`
  - LLM 파싱 실패 시 fallback 데이터 반환
- `backend/app/routers/chat.py` — 신규 라우터 파일 생성 완료
  - `POST /api/chat/stream` — SSE 스트리밍 채팅
  - 요청: `{ messages: [{role, content}], node_context: {label, role, level} }`
  - `build_chat_system_prompt`로 노드 컨텍스트 system message 주입
