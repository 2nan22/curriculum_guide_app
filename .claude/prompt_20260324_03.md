## Task 1: Detailed Learning View

노드 클릭 시 나타나는 상세 학습 패널을 구현해주세요.

### 요구사항
- `NodeDetailPanel.jsx` - 슬라이드인 사이드 패널 (우측)
  - 표시 정보:
    - 노드 제목 및 설명
    - 예상 학습 기간 (`estimatedWeeks`)
    - 학습 미션 목록 (LLM으로 동적 생성)
    - 핵심 개념 요약 (LLM으로 동적 생성)
  - 레벨별 톤앤매너 최적화:
    - Junior: "이 개념부터 시작하세요! 🌱 (쉽고 친근한 설명)"
    - Mid: "실무 적용 관점에서 이해하세요. (실용적 설명)"
    - Senior: "트레이드오프와 설계 관점에서 분석합니다. (전문적 설명)"
  - 패널 열기/닫기 애니메이션 (CSS transition)
- `src/utils/prompts.js` 에 노드 상세 생성용 프롬프트 추가
  - `buildNodeDetailPrompt(node, role, level)` 함수
  - 미션 3개 + 핵심 개념 5개를 JSON으로 반환하도록 프롬프트 설계
- `src/hooks/useNodeDetail.js` - 노드 상세 데이터 페칭 훅
  - 같은 노드 재클릭 시 캐시된 데이터 재사용 (Map 자료구조 활용)

### 커밋 메시지
```
feat: 노드 클릭 상세 패널 및 레벨별 학습 미션 생성 구현

- NodeDetailPanel 슬라이드인 사이드 패널 구현
- Junior/Mid/Senior 레벨별 톤앤매너 다른 설명 생성
- useNodeDetail 훅에서 노드별 LLM 응답 캐싱 처리
```

## Task 2: Smart Recommendation Engine

커리큘럼 키워드 기반 서적/강의 추천 기능을 구현해주세요.

### 요구사항
- `public/data/recommendations.json` - 큐레이션 추천 데이터베이스
  - 구조:
    ```json
    {
      "keywords": {
        "Spring Boot": {
          "books": [
            { "title": "스프링 부트 핵심 가이드", "author": "장정우", "url": "...", "level": ["Junior", "Mid"] }
          ],
          "lectures": [
            { "title": "스프링 입문 - 코드로 배우는 스프링 부트", "platform": "인프런", "instructor": "김영한", "url": "...", "free": true }
          ]
        },
        "시스템 설계": {
          "books": [
            { "title": "가상 면접 사례로 배우는 대규모 시스템 설계 기초", "author": "Alex Xu", "url": "..." }
          ],
          "lectures": [
            { "title": "System Design Interview", "platform": "ByteByteGo", "url": "..." }
          ]
        }
      }
    }
    ```
  - 커버 범위: Backend, Frontend, AI/ML, Common 도메인의 주요 키워드 30개 이상
- `src/utils/recommendationEngine.js`
  - `getRecommendations(nodeLabel, role, level)` - 노드 레이블과 키워드 매칭
  - 퍼지 매칭(부분 문자열 포함 여부)으로 키워드 검색
  - 레벨 필터링 적용
- `RecommendationSection.jsx` - NodeDetailPanel 내 추천 섹션
  - 서적/강의 탭 분리
  - 각 항목: 제목, 저자/강사, 플랫폼, 링크 버튼

### 커밋 메시지
```
feat: 키워드 기반 서적/강의 추천 엔진 구현

- recommendations.json에 도메인별 큐레이션 데이터 30+ 키워드 등록
- recommendationEngine.js로 노드-키워드 퍼지 매칭 구현
- NodeDetailPanel에 서적/강의 추천 섹션 추가
```

## Task 3: AI Tutor Integration

컨텍스트 인식 사이드바 채팅창과 퀵 퀴즈 기능을 구현해주세요.

### 요구사항
- `AiTutorChat.jsx` - 플로팅 사이드바 채팅창
  - 현재 선택된 노드의 컨텍스트를 System Message로 자동 주입
  - 스트리밍 응답 지원 (`llmService.streamChat` 활용)
  - 메시지 히스토리 유지 (세션 내)
  - 채팅창 최소화/최대화 토글
  - 초기 안내 메시지: "현재 '[노드명]'을 학습 중이시군요! 궁금한 점을 질문해보세요."
- `QuickQuiz.jsx` - 노드별 퀵 퀴즈 컴포넌트
  - "퀴즈 생성" 버튼 클릭 시 LLM으로 OX 또는 객관식 문제 3개 생성
  - 정답 제출 후 해설 표시
  - 퀴즈 생성용 프롬프트: `buildQuizPrompt(node, level)` 추가
- `src/context/TutorContext.jsx` - 튜터 채팅 상태 전역 관리
  - 현재 활성 노드, 메시지 히스토리, 채팅창 열림 상태

### 커밋 메시지
```
feat: 노드 컨텍스트 인식 AI 튜터 채팅 및 퀵 퀴즈 구현

- AiTutorChat에서 현재 노드 컨텍스트를 System Message로 자동 주입
- 스트리밍 응답으로 실시간 채팅 응답 표시
- QuickQuiz 컴포넌트에서 LLM 기반 문제 생성 및 해설 표시
```
