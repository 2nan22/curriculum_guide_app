## Task 1: Progress & Persistence

### 요구사항
- `frontend/src/hooks/useProgress.js` 신규 훅
  - `completedNodes: Set<string>` 상태
  - `toggleComplete(nodeId)` — 완료 토글
  - `getCompletionRate(nodes)` — 전체 완료율 (%)
  - `getBranchRate(branch)` — 브랜치별 완료율
- `frontend/src/services/storageService.js` 신규
  - localStorage 키: `aipath_{role}_{level}`
  - `save(role, level, completedNodes)` / `load(role, level)` / `clear(role, level)`
  - 로드맵 재진입 시 자동 복원
- `ProgressBar.jsx` 신규 컴포넌트
  - 전체 진행률: 헤더 우측 고정 표시
  - 브랜치별 미니 진행률: 마인드맵 branch 노드 옆 표시
- `NodeDetailPanel`에 "학습 완료" 체크박스 추가
  - 완료 시 마인드맵 노드 초록색 전환
  - `useProgress` 훅 연동

### 커밋 메시지

## Task 3: Final Quality Check

### 요구사항
- **반응형 레이아웃**
  - 모바일 (`< 768px`): 마인드맵 숨기고 리스트뷰 기본, 노드 상세를 하단 시트로 표시
  - 태블릿 (`768px~1024px`): 마인드맵 축소 + 상세 패널 하단
  - 데스크탑 (`> 1024px`): 기존 좌우 분할 레이아웃
  - Tailwind v4 반응형 클래스 활용 (`sm:`, `md:`, `lg:`)

  - `MindmapNode` → `React.memo()` 적용

  - `NodeDetailPanel` 응답 캐시 (Map) 동작 확인
- **코드 클린업**
  - 미사용 import 제거
  - `.env.example` 최신 상태 유지 확인 (실제 키 노출 없는지 확인)
  - `README.md` 작성: 설치 방법, Docker 실행, Ollama/Upstage 설정 안내
  - `docker-compose.yml` 프로덕션 빌드 프로필 추가 (`--profile prod`)
