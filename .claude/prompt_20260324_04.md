## Task 1: Progress & Persistence

학습 진척도 추적 및 저장 기능을 구현해주세요.

### 요구사항
- `src/hooks/useProgress.js` - 진척도 상태 관리 훅
  - 노드별 완료 상태 토글: `toggleComplete(nodeId)`
  - 전체 완료율 계산: `(완료 노드 수 / 전체 노드 수) * 100`
  - 카테고리(브랜치)별 완료율 계산
- `src/services/storageService.js` - 저장소 추상화
  - localStorage 기본 사용
  - 저장 키: `aipath_progress_{role}_{level}`
  - 저장 구조:
    ```json
    {
      "completedNodes": ["leaf_1_1", "leaf_1_2"],
      "lastUpdated": "2026-03-24T10:00:00Z",
      "role": "Backend",
      "level": "Mid"
    }
    ```
  - (선택) Supabase 연동 시 `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY` 환경변수 사용
- `ProgressBar.jsx` - 진행률 표시 컴포넌트
  - 전체 진행률: 상단 고정 헤더 바
  - 브랜치별 진행률: 마인드맵 브랜치 노드 옆에 소형 표시
- 노드 완료 처리:
  - NodeDetailPanel에 "학습 완료" 체크박스 추가
  - 완료 시 마인드맵 노드 색상 변경 (초록색) + 완료 아이콘 표시

### 커밋 메시지
```
feat: 노드별 학습 완료 상태 및 로컬스토리지 진척도 저장 구현

- useProgress 훅으로 노드 완료 상태 및 완료율 계산
- storageService로 localStorage 기반 진척도 자동 저장/복원
- ProgressBar 컴포넌트 헤더 및 브랜치별 표시
```

## Task 2: UI/UX Refinement

마인드맵 인터랙션과 애니메이션을 완성해주세요.

### 요구사항
- **줌/팬(Zoom/Pan) 인터랙션** - `MindmapLayout.jsx` 개선
  - 마우스 휠로 줌 인/아웃 (0.5x ~ 2x 범위)
  - 마우스 드래그로 캔버스 이동
  - "전체 보기" 버튼으로 초기 뷰 리셋
  - SVG `transform` 속성 활용 (또는 `react-zoom-pan-pinch` 라이브러리)
- **Framer Motion 애니메이션**
  - 패키지 설치: `npm install framer-motion`
  - SelectionPage → RoadmapPage 페이지 전환: 페이드인 + 슬라이드업
  - NodeDetailPanel 등장: 오른쪽에서 슬라이드인
  - 마인드맵 노드 초기 등장: 루트부터 순차적 페이드인 (stagger 0.05s)
  - 노드 완료 시: 초록색으로 색상 전환 애니메이션
- **라우팅 구조 정리** - `react-router-dom` 활용
  - `/` → SelectionPage
  - `/roadmap` → RoadmapPage
  - URL에 선택 파라미터 반영: `/roadmap?role=Backend&level=Mid`

### 커밋 메시지
```

- Framer Motion으로 페이지 전환 및 노드 등장 애니메이션 적용
```
